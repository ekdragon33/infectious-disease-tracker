#  Infectious Disease Tracker

A Python tool that pulls real-time global disease data and performs epidemiological analysis.

## What it does
- Fetches live data from a public health API
- Analyzes death rates across 180+ countries
- Calculates total cases, deaths, and average mortality rates

## Tech Stack
- Python
- Pandas
- Requests

## How to run
```bash
pip install pandas requests
python main.py
```

## Sample Output
- 704 million cases tracked globally
- 7 million deaths analyzed  
- Average death rate: 1.39%
- Yemen highest death rate: 18.07%
